# MSL-FER

A Multi-Scale Learning based Facial Expression Recognition (FER) system built using PyTorch.

## Features

- Facial Expression Recognition
- Multi-Scale CNN
- Attention Module
- Real-Time Webcam Prediction
- FER2013 Dataset
- PyTorch Implementation

## Dataset

FER2013

## Tech Stack

- Python
- PyTorch
- OpenCV
- NumPy
- Matplotlib
